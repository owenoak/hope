Classes:

- Identifying things:

	- class="_reset" 			= reset css properties, applied to most widgets

	- class="_<ClassName>" 		= our base class for each Class (automatically applied)

	- part='x'					= part that we pick up, either to replace or to update value
									(note: on replace, we pick up attributes already on the class)

	- message='x'				= update with message dictionary entry (statically) before drawing
									(handled in the template)



- Flags
	- disabled='true'			= disabled flag (only set when NOT enabled)
	- selected='true'			= selected flag (only set when enabled)
	- expanded='true'			= expanded/collapsed flag
	- visible='true'			= true|false|offscreen
	- round=''					= [S|M|L|H] + [-TL|-TR|-BL|-BR]	
	- showHeader='true'			= show/hide header flag
	- showBody='true'			= show/hide body flag
	- showFooter='true'			= show/hide footer flag


- Common helper classes

	- positioning, etc
		- class='_inline'			= inline block


	- labels and fields
		- class='_title'			= this is a title, generally makes it stand out 
		- class='_label'			= normal label, like form label, etc
		- class='_hint'				= hint, de-emphasized


	- shade/color
		- class='_light'			= lighter variation (eg: lighter)
		- class='_medium'			= normal variation (note: should be assumed if not _light or _heavy)
		- class='_heavy'			= heavier variation (eg: darker)

	- sections
		- class='_section'			= section panel (outside)
		- class='_header'			= header
		- class='_body'				= body
		- class='_footer'			= footer
